# Marketplace Courses MVP

## Descripción

Aplicación marketplace donde instructores crean cursos y estudiantes pueden inscribirse.

Tecnologías:

Backend:
- Node.js
- Express
- MongoDB
- JWT
- Nodemailer
- Cloudinary

Frontend:
- React
- Vite

## Modelos

User
Course
Enrollment
Review
Category
Conversation
Message
Notification
Token

## Funcionalidades

- Registro/login
- Verificación email
- Reset password
- Crear cursos
- Subir imágenes
- Inscribirse en cursos
- Workflow de inscripción

## Instalación

API:

cd api
npm install
npm run dev

Client:

cd client
npm install
npm run dev
